## Product Overview

TaskVault is a cloud-based **project management and team collaboration platform** built for modern software and product teams. It combines task management, real-time messaging, file sharing, and workflow automation into a single workspace.

### Core Capabilities

| Capability | Description |
|---|---|
| **Task Management** | Create, assign, and track tasks with due dates, labels, priorities, and subtasks. |
| **Projects & Boards** | Organize work into projects using Kanban boards, list views, or timeline (Gantt) views. |
| **Team Chat** | Real-time messaging within projects and direct messages between team members. |
| **File Sharing** | Attach files to tasks and projects. Integrates with Google Drive and Dropbox. |
| **Workflow Automation** | Set up rules like "When a task moves to Done, notify #general channel." |
| **Time Tracking** | Built-in timer and manual time entry per task. Available on Pro and Enterprise. |
| **Reporting & Dashboards** | Visual dashboards showing team velocity, task completion rates, and workload distribution. |
| **Integrations** | Native integrations with Slack, GitHub, GitLab, Google Drive, Jira, and Zapier. |
| **Mobile App** | Full-featured iOS and Android apps with push notifications. |
| **API** | RESTful API for custom integrations and automation (Pro and Enterprise plans). |

## Support Hours & SLA Commitments

| Plan | Support Channel | Hours | First Response SLA |

### SLA Details

- **Acknowledgment:** All tickets receive an automated acknowledgment within 2 minutes.
- **First Human Response:**
  - Critical (system down): 1 hour (Enterprise), 4 hours (Pro)
  - High (feature broken): 4 hours (Enterprise), 12 hours (Pro)
  - Medium (how-to / general): 8 hours (Enterprise), 24 hours (Pro)
  - Low (feedback / feature request): 48 hours (all plans)
- **Resolution Target:** 80% of non-critical tickets resolved within 24 hours.
- **Escalation:** Any ticket exceeding SLA is auto-escalated to a Team Lead.
